# Claude Code v2.1.193 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-193

## 💡 注目ポイント

### 1. Auto モードのシェルコマンド分類が強化

`autoMode.classifyAllShell` 設定を追加し、Bash/PowerShell のすべてのコマンドを Auto モードの分類器経由にルーティングできるようにしました。これにより、任意のコード実行パターンだけでなく、すべてのシェルコマンドが安全に処理されます。既存の設定では一部のパターンしか対象になっていなかったため、より広範なコマンドが保護対象になります。

### 2. Auto モード拒否理由の可視化

Auto モードによるコマンド拒否の理由がトランスクリプト、トースト通知、および `/permissions` コマンドで確認できるようになりました。これにより、ユーザーがなぜコマンドが拒否されたのかを明確に理解できるようになり、セキュリティポリシーの遵守がしやすくなります。

### 3. ライブファイルパスオートコンプリート

Bash モード（`!`）でライブファイルパスオートコンプリートが追加されました。これにより、ファイルパスを入力する際の効率が向上し、ミスも減ります。

### 4. MCP サーバー認証の改善

MCP サーバーが認証を必要とする場合、自動的に `/mcp` への指示を表示するようになったほか、`headersHelper` が 401/403 エラー発生時に自動的に再実行および再接続するよう改善されました。これにより、認証関連の問題がよりスムーズに解決できます。

### 5. バックグラウンドタスク管理の改善

バックグラウンドタスクの管理に関する複数の問題が修正されました。特に、セッション移行時のタスクキャンセルやメインターンのバグが修正され、バックグラウンドエージェントの動作がより安定しました。また、バックグラウンドエージェントが「応答を終了してください」というメッセージを表示しなくなったため、他のタスクにも影響を与えずに動作します。
