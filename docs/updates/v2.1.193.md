# Claude Code v2.1.193 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-193

## 💡 注目ポイント

### 1. Autoモードのシェルコマンド分類拡張

`autoMode.classifyAllShell` 設定を追加し、Bash/PowerShell のすべてのコマンドを Autoモードの分類器経由にルーティングできるようにしました。これにより、任意のコード実行パターンだけでなく、すべてのシェルコマンドが安全に処理されます。既存の設定では一部のパターンしか処理されていなかったため、より広範なコマンドが保護対象になります。

### 2. Autoモード拒否理由の可視性向上

Autoモードによるコマンド拒否の理由がトランスクリプト、トースト通知、および `/permissions` コマンドで確認できるようになりました。これにより、ユーザーがなぜコマンドが拒否されたのかを明確に理解できるようになり、セキュリティポリシーの遵守がしやすくなります。

### 3. ライブファイルパスオートコンプリート

Bash モード（`!`）でライブファイルパスのオートコンプリートが追加されました。これにより、ファイルパスを入力する際の効率が向上し、ミスを防ぐことができます。

### 4. MCPサーバー認証通知の改善

MCPサーバーが認証を必要とする場合、起動時に `/mcp` を指す通知が表示されるようになったため、ユーザーは認証が必要なサーバーを簡単に識別できます。

### 5. 背景タスク管理の改善

アイドル状態のバックグラウンドシェルコマンドに対する自動メモリ圧力再収集が追加されました。これにより、メモリ使用量が効率的に管理され、システムのパフォーマンスが向上します。
