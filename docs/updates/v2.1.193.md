# Claude Code v2.1.193 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-193

## 💡 注目ポイント

### 1. Autoモードのシェルコマンド分類拡張

`autoMode.classifyAllShell` 設定を追加し、Bash/PowerShell のすべてのコマンドを Autoモードの分類器経由にルーティングできるようにしました。これにより、以前は「実行可能コードパターンにのみ適用」されていた分類が、任意のコマンドに適用されるようになります。セキュリティポリシーを一貫して適用したい環境で特に有用です。

### 2. Autoモード拒否理由の可視性向上

Autoモードによるコマンド拒否の理由が、トランスクリプト、トースト通知、および `/permissions` の最近の拒否リストに表示されるようになりました。これにより、ユーザーはなぜ特定のコマンドがブロックされたのかを明確に確認でき、セキュリティポリシーの理解と遵守が容易になります。

### 3. ライブファイルパスオートコンプリート

Bash モード（`!`）でライブファイルパスオートコンプリートが追加されました。これにより、ファイルパスを指定する際の入力ミスを減らし、作業効率が向上します。特に大規模なプロジェクトでファイルパスを正確に入力する必要がある場合に効果的です。

## 📋 変更一覧

### ✨ 新機能

| 変更 | 誰にどう嬉しいか |
|---|---|
| `autoMode.classifyAllShell` 設定の追加 | Bash/PowerShell のすべてのコマンドを Autoモードで分類可能に |
| Autoモード拒否理由の表示機能追加 | 拒否されたコマンドの理由を明確に確認できる |
| ライブファイルパスオートコンプリート（Bash モード） | ファイルパス入力を正確かつ効率的に行える |
| MCP サーバー認証が必要な場合の起動通知 | MCP サーバーの認証が必要なことが明確に通知される |
