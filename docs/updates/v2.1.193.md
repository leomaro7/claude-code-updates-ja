# Claude Code v2.1.193 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-193

## 💡 注目ポイント

### 1. Autoモードのシェルコマンド処理が大幅向上

`autoMode.classifyAllShell` 設定を追加し、Bash/PowerShell コマンドをすべて Autoモードの分類器経由で処理できるようにしました。これにより、任意のコード実行パターンだけでなく、すべてのシェルコマンドが安全に評価され、セキュリティリスクが軽減されます。また、Autoモードによる拒否理由がトランスクリプトや `/permissions` コマンドで確認できるようになりました。

### 2. バックグラウンドタスク管理の改善

バックグラウンドタスクの管理が大きく改善されました。セッション移行時のタスクキャンセル問題が修正され、すべてのタスクが新しいセッションに引き継がれるようになります。また、メイン会話のバックグラウンド実行時に不要なサブエージェントが生成される問題も修正され、バックグラウンドエージェントがより安定して動作するようになっています。

### 3. MCP サーバー認証の自動リトライ機能

MCP `headersHelper` の認証が改善され、ツール呼び出しで 401/403 エラーが発生した場合、ヘルパーが自動的に再実行および再接続するようになります。これにより、MCP サーバーの認証問題が発生しても、ユーザー介入なしで自動的に回復できるようになりました。
