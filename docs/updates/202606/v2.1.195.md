# Claude Code v2.1.195 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-195

## 💡 注目ポイント

### 1. `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` の追加 — フルスクリーンモードでのマウス操作を制御

`CLAUDE_CODE_DISABLE_MOUSE_CLICKS` 環境変数を追加し、フルスクリーンモードでのマウスクリック、ドラッグ、ホバーを無効にしながら、ホイールスクロールを維持できます。これにより、特定の使用シナリオに合わせてマウス操作を細かく制御できます。

### 2. ハイフン付き識別子のフックマッチャーの修正 — 正確な一致を実現

ハイフン付き識別子（例：`code-reviewer`、`mcp__brave-search`）を持つフックマッチャーが、誤って部分一致していた問題を修正しました。これにより、正確な一致が可能になりました。すべてのツールにマッチさせるには `mcp__brave-search__.*` を使用してください。

### 3. macOS での音声ディクテーションの改善 — 長時間セッションでのサイレンスのキャプチャを修正

macOS での音声ディクテーションが、デフォルトの入力デバイスが変更された後、長時間セッションでサイレンスをキャプチャする問題を修正しました。これにより、音声入力の安定性が向上します。

### 4. 外部プラグインのインストールコンセントの修正 — プロジェクト設定による有効化で明示的なインストール同意が不要に

プロジェクトの `.claude/settings.json` によって有効化された外部プラグインが、すべてのローダーパスで明示的なインストール同意を必要としないように修正されました。これにより、プラグインの管理が簡素化されます。
