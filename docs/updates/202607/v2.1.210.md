# Claude Code v2.1.210 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-210

## 💡 注目ポイント

### 1. 折りたたまれたツールサマリーラインに経過時間カウンターを追加

長時間実行されるツール呼び出しが、見た目で停止しているように見えないように、経過時間カウンターを追加しました。

### 2. `Write(path)`、`NotebookEdit(path)`、`Glob(path)` の権限ルールに対する起動警告の追加

これらの権限ルールは `Edit(path)` または `Read(path)` を使用するように変更されました。

### 3. `isolation: 'worktree'` サブエージェントがメインリポジトリのチェックアウトに対して git 変換コマンドを実行できる問題を修正

サブエージェントは自身の分離されたワークツリーに対してのみ git 変換コマンドを実行できるようになりました。

### 4. `ultracode` キーワードのオプトインが非人間由来の入力（webhook ペイロードや転送された PR コメントなど）で発火する問題を修正

`ultracode` キーワードのオプトインは人間由来の入力でのみ発火するようになりました。

### 5. クラッシュテレメトリにレンダリングされたテキストフラグメントが漏れる問題を修正

UI コンポーネントがスタイル付きテキスト要素の外にコンテンツを返した場合、レンダリングされたテキストフラグメントがクラッシュテレメトリに漏れる問題を修正しました。
