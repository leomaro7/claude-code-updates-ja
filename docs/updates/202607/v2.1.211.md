# Claude Code v2.1.211 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-211

## 💡 注目ポイント

### 1. サブエージェントのテキストと思考プロセスをストリーム JSON 出力に含める機能の追加

`--forward-subagent-text` フラグと `CLAUDE_CODE_FORWARD_SUBAGENT_TEXT` 環境変数を追加し、サブエージェントのテキストと思考プロセスをストリーム JSON 出力に含めることができるようになりました。これにより、サブエージェントの動作をより詳細に追跡・分析できるようになります。

### 2. バックグラウンドエージェントの結果報告が改善

バックグラウンドエージェントの実行状況をより正確に報告するようになりました。Claude は、まだ実行中のエージェントのステータスを報告し、実際の完了を待つようになりました。これにより、エージェントの実行状況をより正確に把握できるようになりました。

### 3. メモリインデックスの上限警告が改善

メモリインデックスの上限警告が改善され、ロードされたコンテンツのみを測定するようになりました。これにより、不要な警告を減らし、メモリ使用量の管理をより正確に行えるようになりました。

### 4. 整数環境変数の表記が拡張

整数環境変数（タイムアウト、トークンバジェット、リトライ回数など）が科学的表記や数字セパレータのスペリング（`1e6`、`64_000` など）に対応するようになりました。これにより、環境変数の設定がより柔軟になりました。

### 5. プロンプトキャッシュの回帰修正

Bedrock、Vertex、Mantle、Foundry で発生していたプロンプトキャッシュの回帰が修正されました。これにより、トレーリングシステムコンテキストブロックが新規入力トークンとして課金されることがなくなり、コストが適正に計算されるようになりました。
