# Claude Code v2.1.196 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-196

## 💡 注目ポイント

### 1. 組織デフォルトモデルのサポート — 管理者が組織コンソールで設定

管理者は組織コンソールでデフォルトモデルを設定できるようになりました。ユーザーが自分でモデルを選択していない場合、`/model` コマンドで「Org default」または「Role default」として表示されます。

```mermaid
flowchart LR
    Admin[Admin sets default model in org console]
    Admin --> User[User sees "Org default" or "Role default" in /model]
```

これまでは、各ユーザーが個別にデフォルトモデルを選択する必要がありました。この変更により、組織全体で一貫したデフォルトモデルを設定できるようになり、ユーザーの設定作業が簡素化されます。

### 2. セッションの読みやすいデフォルト名 — 開始時に自動生成

セッション開始時に、識別しやすいデフォルト名が自動生成されるようになりました。これにより、セッションを簡単に識別し、メッセージを送信できるようになります。

```mermaid
flowchart LR
    Start[Session Start]
    Start --> DefaultName[Generate Default Name]
    DefaultName --> Identify[Easier to Identify and Message]
```

以前は、セッション名が自動生成されず、ユーザーが手動で名前を付ける必要がありました。この変更により、セッションの管理がより簡単になります。

### 3. チャットでのクリック可能なファイル添付 — Cmd/Ctrl-click で Finder/Explorer でファイルを開く

チャット内でファイル添付をクリックできるようになりました。Cmd/Ctrl-click で Finder または Explorer でファイルを開くことができます。

```mermaid
flowchart LR
    Chat[Chat with File Attachment]
    Chat --> Click[Clickable File Attachment]
    Click --> Open[Cmd/Ctrl-click to Open in Finder/Explorer]
```

以前は、チャット内でファイルを直接開くことができませんでした。この変更により、ファイルの確認がより簡単になります。
