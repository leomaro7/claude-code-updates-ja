# Claude Code v2.1.202 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-202

## 💡 注目ポイント

### 1. ダイナミックワークフローサイズの設定追加

`/config` に「Dynamic workflow size」設定が追加され、小・中・大のエージェント数でワークフローのサイズを制御できるようになりました。これは強制的な上限ではなく、アドバイザリーガイドラインです。

### 2. OpenTelemetry 属性の追加

ワークフロー生成エージェントが発行するテレメトリに `workflow.run_id` と `workflow.name` の OpenTelemetry 属性が追加され、ワークフロー実行のアクティビティを OTel データから再構築できるようになりました。

### 3. `/review <pr>` の変更

`/review <pr>` は高速なシングルパスレビューに戻り、マルチエージェントレビューは `/code-review <level> <pr#>` で選択した努力レベルで実行するようになりました。
