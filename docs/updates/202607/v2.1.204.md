# Claude Code v2.1.204 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-204

## 💡 注目ポイント

### 1. SessionStart フック中のイベントストリーミング修正

ヘッドレスセッションで SessionStart フック中にフックイベントがストリーミングされず、リモートワーカーがフック中にアイドル状態で解放される問題を修正しました。

```mermaid
sequenceDiagram
    participant H as Hook
    participant S as Session
    participant W as Worker
    S->>H: SessionStart
    H->>W: Event Streaming
    note right of W
        以前はストリーミングされず、
        ワーカーがアイドル解放される可能性あり
    end note
    H->>W: Event Streaming (修正後)
```

この修正により、SessionStart フック中のイベントが正しくストリーミングされるようになり、リモートワーカーが意図しないタイミングでアイドル解放されることがなくなります。
