# Claude Code v2.1.208 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-208

## 💡 注目ポイント

### 1. スクリーンリーダーモードの追加 — 視覚障害者ユーザーのアクセシビリティ向上

スクリーンリーダーユーザー向けに、プレーンテキストレンダリングモードをオプトインで追加しました。`claude --ax-screen-reader` を実行するか、`CLAUDE_AX_SCREEN_READER=1` を設定するか、設定に `"axScreenReader": true` を追加することで有効化できます。これにより、スクリーンリーダーユーザーがより快適に利用できるようになります。

### 2. `vimInsertModeRemaps` 設定の追加 — Vim モードでの挿入モードマッピングのカスタマイズ

Vim モードで、`jj` のような 2 キーの挿入モードシーケンスを Escape にマッピングする設定を追加しました。これにより、Vim ユーザーがより快適に利用できるようになります。

### 3. `CLAUDE_CODE_PROCESS_WRAPPER` の追加 — 企業内ランチャーのサポート

エージェントビューとバックグラウンドサービスが、指定されたラッパー実行可能ファイルを通じて Claude Code を自己起動するようになりました。これにより、企業内のランチャーを尊重し、セキュリティを強化できます。

### 4. フルスクリーンモードでのマウスクリックサポートの追加 — 操作性の向上

フルスクリーンモードで、マルチセレクトメニューや「その他」入力行でのマウスクリックをサポートするようになりました。これにより、操作性が向上します。

### 5. メモリリークとパフォーマンスの改善 — 長時間セッションの安定性向上

複数のメモリリークを修正し、パフォーマンスを改善しました。これにより、長時間セッションの安定性が向上します。
