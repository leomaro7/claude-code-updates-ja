# Claude Code v2.1.207 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-207

## 💡 注目ポイント

### 1. Auto モードがデフォルトで有効に

Bedrock、Vertex AI、Foundry で Auto モードがデフォルトで有効になりました。これにより、ユーザーは `CLAUDE_CODE_ENABLE_AUTO_MODE` のオプトイン設定をしなくても自動で適切なモデルが選択されるようになります。不要な場合は `disableAutoMode` 設定で無効化できます。

### 2. ターミナルフリーズとキーストロークの遅延の修正

非常に長いリスト、テーブル、段落、コードブロックを含むストリーミングレスポンス時にターミナルがフリーズし、キーストロークが遅延する問題が修正されました。これにより、長いレスポンスを扱う際の操作性が向上します。

### 3. リモート管理設定のセキュリティコンセントダイアログ表示修正

非対話型実行（`claude -p`、SDK）からリモート管理設定が永続的に同意済みとして記録される問題が修正されました。これにより、セキュリティコンセントダイアログが適切に表示されるようになります。

### 4. 不正なプロンプトインジェクション警告の修正

無害なシステム生成の会話更新によって引き起こされる不正なプロンプトインジェクション警告が修正されました。これにより、誤った警告による混乱が減少します。

### 5. クラッシュループの修正

エージェントチームで不正なチームメイトのメールボックスメッセージが原因で発生するクラッシュループが修正されました。これにより、エージェントチームの安定性が向上します。
