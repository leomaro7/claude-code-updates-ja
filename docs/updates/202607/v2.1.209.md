# Claude Code v2.1.209 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-209

## 💡 注目ポイント

### 1. `/model` およびその他のダイアログが `claude agents` バックグラウンドセッションでブロックされないように修正

以前の過度に広範なガードを元に戻し、`claude agents` バックグラウンドセッションで `/model` およびその他のダイアログがブロックされないようにしました。これにより、バックグラウンドセッションでこれらのダイアログを使用できるようになりました。
