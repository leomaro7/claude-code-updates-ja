# Claude Code v2.1.205 アップデートまとめ

> 出典: https://code.claude.com/docs/en/changelog#2-1-205

## 💡 注目ポイント

### 1. セッショントランスクリプトファイルの改ざんをブロックする自動モードルールの追加

セッショントランスクリプトファイルの改ざんを防ぐための自動モードルールが追加されました。

### 2. `--json-schema` の修正

無効なスキーマの場合、`--json-schema` がサイレントで非構造化出力を生成する問題と、`format` キーワードを使用するスキーマが拒否される問題が修正されました。

### 3. バックグラウンドエージェントのステータス表示の改善

バックグラウンドエージェントが `SendMessage` で再開された後も「失敗」または「完了」として表示される問題や、エージェントのターンに読み取り可能なテキストが含まれていない場合に「入力が必要」から「作業中」に戻る問題が修正されました。

### 4. 自動モードの改善

`rm -rf` コマンドを実行する前に、コンテキストから解決できない変数に対して確認を求めるよう自動モードが改善されました。

### 5. バックグラウンドタスク通知の改善

バックグラウンドタスク通知が、人間の入力が発生していないことを明示的に示すようになり、トランスクリプト内の捏造された承認が実行されることを防ぎます。
